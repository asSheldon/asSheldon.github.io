# CSS布局分为两栏布局以及三栏布局

# 三栏布局--双飞翼布局

双飞翼布局分为header、body、left、right、footer
body属于自适应空间，left和right属于固定区域

具体思路：
在左右中元素都变成浮动元素，向左浮动，之后左元素使用margin-left:-100%，这里的margin负值的意思是向左移动上一个浮动元素的总长，右元素使用margin-left:-220px,这里的-220是右元素的长，也就是向左移动220px并覆盖上一个移动元素的220px

*{margin: 0;padding: 0;}
body{min-width: 700px;}
.header{
    text-align: center;
}
.left,.main,.right{
    float: left;
    min-height: 130px;
}
.left{
    margin-left: -100%;
    width: 200px;
    background: red;
}
.right{
    margin-left: -220px;
    width: 220px;
    background: blue;
}
.footer{
    border: 1px solid #333;
    background: #aaa;
    text-align: center;
    clear: both;
}
.main{ 
    width: 100%;
}
.main-inner{ 
    margin-left: 200px;
    margin-right: 220px;
    min-height: 130px;
    background: green;
    word-break: break-all;
}
<body>
    <div class="header">
        <p>header</p>
    </div>
    <div class="main">
        <div class="main-inner">
            <h4>main</h4>
            <p>HHHHHHHHHHHHHHHHHHHHHH
            hhhhhhhhhhhhhhhhhhhhhhhhhhhhhhh
            HHHHHHHHHHHHHHHHHHHHHH
            hhhhhhhhhhhhhhhhhhhhhhhhhhhhhhh
            </p>
        </div>
    </div>
    <div class="left">
        <p>left</p>
    </div>
    <div class="right">
        <p>right</p>
    </div>
    <div class="footer">
        <p>footer</p>
    </div>
</body>

# 三栏布局--圣杯布局

具体思路：
圣杯布局与双飞翼布局有所不同，在布局方面，将中间的部分通过padding空出左右两个部分，并且在圣杯布局是将中部main部分分成三部分，分别是mid,left,right 同时也将这三部分设置成向左浮动元素，最后通过position相对位置和margin-left:-100%，最后通过right来将左元素相对于自己原原来在其右边来将元素放到之前 布置好的位置，反之右元素也一样。

.header{
    text-align: center;
}
.content{
    padding-left: 200px; 
    padding-right: 150px;
}
.content .mid,.content .left,.content .right{
    float: left;
    min-height: 130px;
}
.content .mid{
    width: 100%;
    background: green;
}
.content .left{
    position: relative;
    margin-left: -100%;
    width: 200px;
    background: red;
    right: 200px;
}
.content .right{
    position: relative;
    left: 150px;
    margin-left: -150px;
    width: 150px;
    background: blue;
}
.footer{
    border: 1px solid #333;
    background: #aaa;
    text-align: center;
    clear: both;
}

<body>
    <div class="header">
        header
    </div>
    <div class="content">
        <div class="mid">11</div>
        <div class="left">22</div>
        <div class="right">33</div>
    </div>
    <div class="footer">
        footer
    </div>
</body>

# 两栏布局

具体思路：
左边元素固定，右边元素自适应

# 使用float实现

/* 清除浏览器默认边距 */
* {
    margin: 0;
    padding: 0;
}

.wrap {
    overflow: hidden;
    border: 1px solid red;
}
/* 脱离文档流 */
.left {
    float: left;
    width: 200px;
    height: 200px;
    background: purple;
}

.right {
    margin-left: 200px;
    background: skyblue;
    height: 200px;
}

# 使用绝对定位实现

/* 清除浏览器默认边距 */
* {
    margin: 0;
    padding: 0;
}

.wrap {
    overflow: hidden;
    position: relative;
}
/* 脱离文档流 */
.left {
    position: absolute;
    left: 0;
    top: 0;
    width: 200px;
    height: 200px;
    background: purple;
}

.right {
    margin-left: 200px;
    background: skyblue;
    height: 200px;
}

# 使用弹性布局实现

/* 清除浏览器默认边距 */
* {
    margin: 0;
    padding: 0;
}
.wrap {
    display: flex;
}
.left {
    height: 200px;
    background: purple;
    flex:0 0 200px
}
.right {
    background: skyblue;
    height: 200px;
    flex: 1;
}

# 使用表格布局实现

/* 清除浏览器默认边距 */
* {
    margin: 0;
    padding: 0;
}
/* 表格布局 */
.wrap {
    display: table;
    width: 100%;
}

.left {
    display: table-cell;
    width: 200px;
    height: 200px;
    background: purple;
}

.right {
    display: table-cell;
    background: skyblue;
    height: 200px;
}
